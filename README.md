## Installation

[Installation documentation](https://raiden-client-python.readthedocs.io/en/latest/installation.html)

raiden-client run on Python 3.5+

Install from pypi:

```
$ pip install -U raiden-client
```

Or, clone repo and run:
```
$ pip install .
```
